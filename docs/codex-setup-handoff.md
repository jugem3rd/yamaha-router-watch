# Codex Setup Handoff

This repository contains the Yamaha Router Watch MVP.

The initial setup should preserve `README.md` and focus on the Cloudflare Workers API, Cloudflare D1 schema, local test scripts, and design documents.

Out of scope for the initial setup:

- Yamaha router hardware integration.
- Production Resend notification logic.
- Full Cloudflare Pages dashboard.
- Customer authentication and billing.
- Raw syslog forwarding.
- Full `show config` collection.
